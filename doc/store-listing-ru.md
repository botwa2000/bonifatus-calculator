# Store Listing — Русский (ru)

## Play Store

**Название приложения** (макс. 50 символов):
Bonifatus – Оценки и Награды для Семьи

**Краткое описание** (макс. 80 символов):
Отслеживайте оценки, зарабатывайте бонусы и превращайте успехи в школе в награды.

**Полное описание** (макс. 4000 символов):
Bonifatus делает школьные оценки значимыми для всей семьи.

Ученики фотографируют свои оценки телефоном – приложение читает их автоматически и начисляет бонусные баллы в зависимости от успеваемости. Родители устанавливают награды. Выигрывают все.

**Для учеников:**
• Мгновенно фиксируйте оценки камерой телефона (OCR)
• Наблюдайте, как растут ваши бонусные баллы с каждой хорошей оценкой
• Используйте Калькулятор оценок для предварительного расчёта среднего балла
• Отслеживайте прогресс со временем во вкладке «Аналитика»

**Для родителей:**
• Обзор всех оценок детей в режиме реального времени
• Настройка бонусных коэффициентов для поощрения отличников
• Расчёт баллов и выдача наград в удобном темпе
• Настройка еженедельного отслеживания текущих оценок

**Система оценивания:**
Поддерживает немецкую систему оценок (1–6) с настраиваемыми множителями:
• Уровень 1 (оценки 1–2): бонус 2,0×
• Уровень 2 (оценка 3): бонус 1,5×
• Уровень 3 (оценка 4): бонус 1,0×

**Конфиденциальность:**
Без рекламы. Без отслеживания. Данные вашей семьи хранятся на наших защищённых серверах в Европе.

## App Store

**Название** (макс. 30 символов):
Bonifatus

**Подзаголовок** (макс. 30 символов):
Оценки и Награды для Семьи

**Рекламный текст** (макс. 170 символов):
Каждая хорошая оценка становится наградой. Ученики фиксируют оценки, зарабатывают баллы – родители выбирают призы. Школьный успех становится интересным.

**Ключевые слова** (макс. 100 символов):
оценки,школа,награда,бонус,баллы,ученики,родители,калькулятор,семья,обучение,трекер

**Описание** (макс. 4000 символов):
Bonifatus создаёт связь между школьной успеваемостью и семейной мотивацией.

Ученики используют камеру для фиксации оценок – технология OCR считывает их автоматически. Бонусные баллы рассчитываются мгновенно на основе настраиваемых уровневых коэффициентов. Родители видят всё в режиме реального времени и устанавливают награды.

**Функции для учеников:**
• Фиксация оценок камерой с автоматическим OCR
• Расчёт бонусных баллов с уровневыми множителями
• Калькулятор оценок: добавляйте предметы, смотрите средний балл
• Аналитика: история бонусных баллов и распределение оценок
• Заметки: отслеживайте текущие оценки между четвертями

**Функции для родителей:**
• Обзор всех подключённых детей в реальном времени
• Настраиваемые уровни бонусов (2×, 1,5×, 1×)
• Проверка быстрых оценок и расчёт одним нажатием
• Аналитика со средними баллами и тенденциями по каждому ребёнку
• Связь через QR-код для безопасного подключения детей

**Как это работает:**

1. Родители создают аккаунт и настраивают систему бонусов
2. Дети подключаются через сканирование QR-кода
3. Ученики фотографируют оценки – приложение их считывает
4. Бонусные баллы накапливаются автоматически
5. Родители проверяют, одобряют и выдают награды

Без рекламы. Без подписки. Данные хранятся на защищённых европейских серверах.
